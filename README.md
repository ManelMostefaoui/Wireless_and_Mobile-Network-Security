# Wireless and Mobile Network Security Labs

This repository contains laboratory reports and practical exercises completed as part of the **Wireless and Mobile Network Security** course.

The labs explore security challenges in wireless and mobile environments, focusing on network protection, vulnerability assessment, threat analysis, and secure communication practices.

## Skills Demonstrated

* Wireless Network Security
* Mobile Network Security
* Security Assessment
* Threat Analysis
* Secure Network Design
* Defensive Security Techniques

## Laboratory Reports

This repository includes practical work related to wireless and mobile network security concepts, tools, and methodologies.
